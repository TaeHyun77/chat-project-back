# Integrated Queueing System

Java + Spring Boot 기반 공공 API 기반 인천공항 정보 제공 및 STOMP 기반 채팅 프로젝트

## 응답 언어 지침

- 모든 결과값, 설명, 주석, 커밋 메시지, PR 설명 등은 반드시 한글로 작성한다.
- 코드 내 변수명, 함수명, 클래스명 등 식별자는 영문을 유지한다.

## 빌드 및 실행

```bash
./gradlew build
./gradlew test
```
